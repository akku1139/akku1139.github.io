[前の記事](/blog/inspiron-upgrade-1/)でCPUとメモリを交換しようと検討していました。

実際かなり前に交換は終了したのですが、記事を書いていませんでした。

最終的にどのような構成になったのでしょうか?

計画では

```yaml
CPU: Core i5-4670
メモリ: 16GB (8GB *2) DDR3
```

となる予定でした。

実際にハードオフで買ったパーツは

```yaml
CPU: Core i7-4770S (4400 yen)
メモリ: 16GB (8GB *2) DDR3 (2200 yen)
```

の計6600円です。 微妙に高いですね...。

交換して元気に稼働しています。
